# Changelog

## 1.2.5 - 2026-05-19

Applies to:

- `firstaid-1.2.5+fabric1.21.1`
- `firstaid-1.2.5+fabric26.1`
- `firstaid-1.2.5+neoforge1.21.1`
- `firstaid-1.2.5+neoforge26.1`
- `firstaid-1.2.5+forge1.20.1`
- `firstaid-1.2.5+fabric1.21.11-legacy`
- `firstaid-1.2.5+neoforge1.21.11-legacy`

### Added

- Added `/firstaid commandtips on|off` for operators to enable or disable the FirstAid admin command tip shown when joining a world.
- Added matching persistent config-file support through `commandTipsEnabled`, defaulting to `true`.
- Added the new command to the join-time admin command tip so operators can discover how to hide the message.

### Changed

- Ported the 1.2.4 treatment-prompt cleanup to the legacy Fabric and NeoForge 1.21.11 branches so treatment feedback stays centered instead of duplicating actionbar countdowns.
- Marked the `todos.md` Planned Features items as complete.

### Notes

- All seven loader/version modules were bumped to loader-specific `1.2.5` release strings.
