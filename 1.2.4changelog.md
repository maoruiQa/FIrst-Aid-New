# Changelog

## 1.2.4 - 2026-05-09

Applies to:

- `firstaid-1.2.4+fabric1.21.1`
- `firstaid-1.2.4+fabric26.1`
- `firstaid-1.2.4+neoforge1.21.1`
- `firstaid-1.2.4+neoforge26.1`
- `firstaid-1.2.4+forge1.20.1`

### Changed

- Removed duplicate treatment countdown feedback so the centered treatment prompt stays the primary item-use indicator.
- Bumped maintained module versions to loader- and Minecraft-specific `1.2.4` release strings.

### Fixed

- Fixed stale per-limb max health values loading from saved data so current config changes apply to existing players.
- Removed the old per-limb health config validation cap of `12`, allowing larger configured limb health values to load from config files.

### Notes

- The legacy `1.21.11` Fabric and NeoForge lines on this branch also received the limb health config loading and validation updates.
